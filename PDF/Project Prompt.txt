Mini Project
-------------
Ai Integrated Washing Machine

We Already have a semi Automatic Washing Machine 
we have converted it into a fully automated AI Integrated Washing Machine 

We will scan Each Cloth Type Using an App We build that scans the cloth With the camera built on the 
washing machine and it will scan the Cloth type using Gemini API and Give Results Back to the App and thereby to 
the arduine connected to the washing machine . then it will analyse how much water , detergent and time it needs to
do washing and it will inform the washing machine and it does the work Which will Increase the lifetime of cloth

We have connected a relay to the washing machine motor which enables the motor to rotate clock wise for 5 seconds and stop for 2 seconds 
and then rerotate for 5 seconds counter clockwise and it loop on
we have connected Arduino to the Motor which will dispense the amount of detergent to the washing machine as Required  
we have Created a linear regression function to calculate the amount of time the valve must be open to dispense the required volume of
detergent 
(First We calculated the volume of detergent that dispense in specific time, 
That is We found volume of detergent at the time 50ms,100ms,150ms,....upto 2400ms and created an Excel sheet table 
then we calculated the value of a and b of the linear regression equation and found the equation
then we applied the equation to the Arduino code to get the exact amount of dispensed detergent )
