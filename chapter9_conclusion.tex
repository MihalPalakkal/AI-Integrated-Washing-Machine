\section*{Project Summary}
The Intelligent Laundry System successfully transformed a conventional semi-automatic washing machine into a fully automated, AI-integrated smart appliance. The project achieved all its primary objectives:

\begin{itemize}
    \item \textbf{AI-Powered Fabric Recognition:} The Gemini API successfully identifies fabric types, fiber categories, and soil levels from garment images with high confidence scores (average 0.92).
    \item \textbf{Automated Parameter Calculation:} The dual-API architecture accurately determines optimal washing parameters including detergent quantity, water level, soak time, spin duration, and temperature based on fabric analysis.
    \item \textbf{Precision Hardware Control:} The Arduino-controlled system executes wash cycles with exact timing, including programmed motor rotation patterns and linear regression-based detergent dispensing.
    \item \textbf{Linear Regression Calibration:} Empirical data collection and analysis yielded an accurate mathematical model (Volume = 0.022176 × Time - 3.569687) with \(R^2 = 0.997\), ensuring precise detergent delivery.
    \item \textbf{User-Friendly Interface:} The React Native mobile application provides intuitive controls for scanning, monitoring, and manual override, with positive feedback from usability testing.
\end{itemize}

The project achieved several key performance metrics that validate its effectiveness and precision. The system reached an average fabric classification accuracy of 89\% across a diverse range of material types, demonstrating the robustness of the Gemini-based visual analysis. Hardware performance was equally impressive, with detergent dispensing achieving a high precision of $\pm$0.72 ml mean error through the linear regression calibration model. The end-to-end system response time remained efficient, taking under 5 seconds from the initial garment scan to the generation of washing parameters. Most importantly, the project proved the feasibility of full hardware automation and successful retrofit integration, enabling a conventional legacy machine to be upgraded to a smart appliance at a minimal cost.

\section*{Significance}
This project demonstrates that artificial intelligence can be practically integrated into household appliances through cost-effective retrofitting. The ``Eyes and Brain'' architecture—separating visual analysis from decision logic—provides a scalable template for smart appliance development.

The system eliminates user guesswork, extends garment lifespan through appropriate care, and reduces environmental impact by optimizing water and detergent usage. For users, it offers convenience and peace of mind; for the environment, it conserves resources; for researchers, it provides a documented framework for AI-hardware integration.

\section*{Closing Remarks}
The Intelligent Laundry System proves that ``dumb'' appliances can be given intelligence through thoughtful integration of modern technologies. By combining computer vision, embedded systems, and mobile applications, this project creates tangible value from theoretical concepts. The system is fully functional, tested, and ready for real-world deployment, achieving its goal of making laundry care smarter, safer, and more sustainable.
